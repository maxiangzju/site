WebGL Metaball Animation

This example demonstrates the WebGL metaball header animation. The organic blob shapes merge smoothly as they move, creating a fluid, lava lamp-like effect.

Features:

  • Two layered blob effects for depth
  • Smooth polynomial falloff for organic merging
  • Animated wavy boundary at the bottom
  • Hover interaction reveals red accent blobs
  • Dark mode support (try the toggle button)
  • Performance optimized (~30fps, visibility-aware)

Hover over the header to see the accent color effect. The red blobs fade in with staggered timing.
